Welcome to Dianoga's Trash Handler

Purpose

Our team's purpose is to create a mobile application that raises awareness of recycling and detects materials using artificial intelligence. We want to help users improve their recycling habits. In order to create a successful application, we need pictures of recyclable/non-recyclable items.

How to Participate

If want to participate, please go the Data Collection link in the menu above.

We have progress bars to show how close we are to our goals. Please try to upload images in the categories where we are furthest from our goal. Don't forget to select the category label from the dropdown menu before submitting.

Thank You!!!
